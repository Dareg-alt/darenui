# Toast

Toast is a message that is displayed temporarily on the screen. It is used to
display a message to the user.

<Code
  code={`import { Toast, ToastViewport, ToastProvider } from '@daren/ui-components'`}
/>

  <div className="p-8">

  </div>
</Preview>

---

## Properties

<Properties>
  <Property name="title" type="string" required>
    The title of the toast.
  </Property>
  <Property
    name="type"
    type="[default, warning, danger, success, info]"
    required
  >
    The type of the toast.
  </Property>
  <Property name="children" type="[React.ReactNode]" required>
    The content of the component.
  </Property>
</Properties>

---

## Configuration

### ToastProvider

This needs to be wrapped around the root component of your application.

<Code
  code={`<ToastProvider>
  <App />
</ToastProvider>`}
/>

### ToastViewport

This needs to be placed in the root component of your application.

<Code code={`<ToastViewport />`} />
