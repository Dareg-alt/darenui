# Field

Field is an [Input](/input) with a label, description, and error message.

<Code code={`import { Field } from '@daren/ui-components'`} />

## Preview

<Preview
  code={`<Field name="preview-field" label="label" description="description field" />`}
>
  <div className="space-y-2">
    <Field name="preview-field" label="label" description="description field" />
  </div>
</Preview>

---

## Properties

<Properties>
  <Property name="name" type="string" required>
    The identifier of the input.
  </Property>
  <Property name="label" type="string">
    The label of the input.
  </Property>
  <Property name="description" type="string">
    The description of the input.
  </Property>
  <Property name="error" type="string">
    The error message of the input.
  </Property>
</Properties>

---

## error

When defining an error, the input will be marked as invalid.

<Preview
  code={`<Field label="example label" error="this is an example error" />`}
>
  <div className="space-y-2">
    <Field label="example label" error="this is an example error" />
  </div>
</Preview>

## children

children can be passed to the input to add additional elements to the input.
Example: a button.

<Preview
  code={`<Field
  label="example label"
  description="example description"
  error="example error"
>
  <Button size="sm">Browse</Button>
</Field>`}
>
  <div className="space-y-2">
    <Field
      label="example label"
      description="example description"
      error="example error"
    >
      <Button size="sm">Browse</Button>
    </Field>
  </div>
</Preview>
